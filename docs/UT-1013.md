# UT-1013: Corrección Asistida con IA

| Metadato | Detalle |
| :--- | :--- |
| **Responsable** | Analía González (P1) |
| **Estimación** | 20 Horas |

## Descripción
Interfaz para agilizar la calificación de preguntas abiertas (Texto libre).

## Flujo de Trabajo
1. Profesor accede a "Resultados" -> "Corregir".
2. El sistema muestra la respuesta del alumno.
3. **IA Integration:** El sistema muestra una "Sugerencia" y una "Nota propuesta".
4. Profesor puede aceptar la nota o sobrescribirla manualmente.
5. Profesor debe poder dejar un comentario de retroalimentación (Feedback).
