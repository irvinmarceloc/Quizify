# UT-1010: Variación de Preguntas con IA

| Metadato | Detalle |
| :--- | :--- |
| **Responsable** | Irvin Cardozo (P2) |
| **Estimación** | 9 Horas |

## Descripción
Herramienta para evitar que los profesores se queden sin ideas o para generar versiones "Tema A / Tema B".

## Funcionalidad
1. Botón "✨ IA" en el formulario de creación de pregunta.
2. Al hacer clic, simular llamada a API (delay 1-2s).
3. Reemplazar el texto del enunciado con una variante parafraseada.
