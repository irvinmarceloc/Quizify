# UT-1041: Autoría Compartida

| Metadato | Detalle |
| :--- | :--- |
| **Responsable** | Analía González (P1) |
| **Estimación** | 20 Horas |
| **Sprint** | 2 |

## Descripción
Permitir que un profesor (Propietario) invite a otros profesores (Colaboradores) a editar un cuestionario existente.

## Requisitos Funcionales
1. **RF-01:** El sistema debe mostrar un botón "Compartir" en la tarjeta de cada cuestionario.
2. **RF-02:** Al hacer clic, se abrirá un Modal.
3. **RF-03:** El modal debe validar que el input sea un email con dominio institucional (@quizify.com o similar).
4. **RF-04:** Se debe listar los usuarios que ya tienen acceso.

## Criterios de Aceptación (DoD)
- [ ] El modal se abre y cierra correctamente.
- [ ] No se permite invitar correos inválidos.
- [ ] La lista de colaboradores se actualiza sin recargar la página (AJAX/Local update).
